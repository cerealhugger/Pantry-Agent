# PantryAgent Demo Assets

Place each source file in its assigned folder and keep the filenames below.

## Privacy

Everything placed under `people/` is ignored by Git except the placeholder instructions.
The live `manifest.json` is also ignored because generation may add names, voice IDs, and
private asset URLs. Do not force-add these files.

The public `manifest.example.json`, product recordings, brand assets, render outputs, and
project documentation remain eligible for Git unless they contain private information.

## People

Each member folder should contain:

- `portrait.jpg`: front-facing portrait, shoulders visible, even lighting, at least 1024 px.
- `voice-sample.wav`: 20-30 seconds, one clean voice, no music or echo.
- `final-read.wav`: an engaging read of that member's final script section.
- `consent.txt`: a short confirmation that the member authorizes this demo to use their portrait and cloned voice.

## Recordings

Add these 1920x1080, 30 fps screen recordings:

- `01-receipt-to-pantry.mp4`
- `02-recipe-import.mp4`
- `03-plan-shop-eat.mp4`

Do not include API keys, session tokens, notifications, or private account information.

## Brand

Add `logo.png`, preferably a transparent high-resolution PNG.
